# MG - Memory Game

## What this is?
This is a fantastic memory game, or almost it...

## Install

### [NPM](https://www.npmjs.com/)
My case, on debian: `apt install npm`.

### [Parcel](https://parceljs.org/)
My case, on npm: `npm install -g parcel-bundler`.
